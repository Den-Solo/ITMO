Задача напоминает задачу о размене монет.

1) предпросчитаем все возможные значения слагаемых суммы (квадраты чисел от 1 до N)
2)создадим массив a[] размера (N+1), где для каждого индекса будем хранить минимальное количество "квадратных свидетельств" для набора такой суммы.
При N == 0: a[0] = 0
При N == 1: a[1] = 1
остальные значения заполнены +inf.
3) проходимся по суммам  i  = 2 до n включительно. Минимальное количество "квадратных свидетельств" в таком случае для каждой суммы будет равно минимуму из текущего значения (+inf по умолчанию) и минимального из количеств "квадратных свидетельств" для сумм, равных i - j, где j - это возможные слагаемые, меньшие i
