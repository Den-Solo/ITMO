1) рассчитаем, какой массы не хватает (weight_diff)
2) требуется определить уникальное множество карт для набора это массы

массив cnt_cards[0..weight_diff][0..N] - хранит количество карт для набора данной массы с учетом последней карты

массив used[0..weight_diff][0..N] - устанавливает, была ли задействована карта при наборе данной суммы

Проходимся по всем массам [1..weight_diff] и для каждой пытаемся подобрать набор подходящих карт.

cnt_card[w][i + 1] = cnt_cards[w][i] + cnt_cards[w - card_weights[i]][i];

если в выражении cnt_cards[w][i] != 0, значит набор данной массы можно сделать несколькими способами.
если cnt_cards[w - card_weights[i]][i] ==0, то такими картами нельзя собрать массу, 
если cnt_cards[w - card_weights[i]][i] > 1, то вариантов несколько.
если == 1, то вариент есть и только один.

*заполнение такое, что при cnt_card[w][i], i = [0..N] каждый элемент равен max( cnt_card[w][i],cnt_card[w][i - 1])


одновременной фиксируем, какие карты использовались: только если набор существовал и вариант был только один.

3) проходимся по массиву used[weight_diff][0..N] и выводим карты, которые были задействованы.
